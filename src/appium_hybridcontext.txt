**********************************HYBRID CONTEXT**********************************
- hybrid context: when a native app is embedded webview layout in it
- implement or abstract class cannot use with new keyword, except the user create a override 
method for implement or abstract class
- Cannot use Appium Inspector to get selector on web view
+ There are 2 ways:
	- Open remote debugging webview: chrome://inspect
	- Open direct web is embedded to mobile
		Ex: https://webdriver.io/
Download the chromedriver and point the downloaded path in the code: https://stackoverflow.com/a/66621541
+ Need check false negative case when checking with container as Map, List...
+ Alt + Insert to generate code
+ appium --allow-insecure chromedriver_autodownload to auto get chrome driver properly project
